# Calculadora Simples

Calculadora em Python com interface no terminal. Suporta as operações básicas e algumas avançadas.

## Operações Suportadas

- **1** - Soma (+)
- **2** - Subtração (-)
- **3** - Multiplicação (×)
- **4** - Divisão (/)
- **5** - Potência (**)
- **6** - Raiz quadrada
- **7** - Resto da divisão (%)
- **8** - Divisão inteira (//)

## Como usar

```bash
python calculadora.py
```

### Dicas

- Pressione **Enter** ou digite **=** para usar o último resultado calculado
- Digite **0**, **s** ou **sair** para encerrar o programa

## Requisitos

- Python 3.x (usa o módulo `math` da biblioteca padrão)
